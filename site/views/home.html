Spring Data Mock

Spring Data Mock is a framework for mocking Spring Data repositories.

Download

You can download this framework via:

  • Github

  • Maven Central by adding this dependency to your pom.xml file:

    <dependency>
    <groupid>com.mmnaseri.utils</groupid>
    <artifactid>spring-data-mock</artifactid>
    <version>${latest-version}</version>
    <scope>test</scope>
</dependency>

  • By cloning the project and installing it locally:

    git clone https://github.com/mmnaseri/spring-data-mock.git
cd spring-data-mock
mvn install

    and then adding the same dependency as above, or just grabbing the JAR file locally from the target folder.

Building the Code

To make the code builds universal and canonical, I have a Docker configuration attached to this project which installs OpenJDK 8 on Ubuntu Xenial. This is the build environment I will be using to test and release the code.

docker build -t spring-data-mock:jdk8 .
docker run -it --rm -v $(pwd):/src spring-data-mock:jdk8

Then, from inside the Docker image, you can build and test the project with Maven:

cd spring-data-mock
mvn test
